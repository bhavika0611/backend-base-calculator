

# 🧮 Streamlit Calculator App

A simple yet interactive calculator built using **Streamlit** and **Python**. This project demonstrates how basic mathematical operations can be turned into a clean web app using Streamlit.

---

## 🚀 Features

* Addition
* Subtraction


## 🛠️ Technologies Used

* Python
* Streamlit
* Basic Python functions
* Simple UI design with Streamlit components

## ⚙️ Installation & Setup

Follow these steps to run the project on your system:

### 1. Clone the repository

```bash
git clone https://github.com/your-username/streamlit-calculator.git
```

### 2. Open the project folder

```bash
cd streamlit-calculator
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Streamlit app

```bash
streamlit run app.py
```

---

## 💡 How It Works

The app takes two numbers as input and lets the user choose the mathematical operation. Once the operation is selected, the result is displayed instantly on the screen.
